# AI Services

FastAPI service area for InsureFlow AI intelligence features.

## Current Status

Implemented services:

- `triage-service` for rule-based triage and optional local ML inference.
- `document-intelligence-service` for offline LLM-style extraction, missing-document checks, summaries, prompt validation, and audit.

## Planned Responsibilities

- Rule-based claim triage.
- ML model inference for severity and fraud risk.
- LLM document extraction and claim summarization.
- RAG-based adjuster assistance.
- AI response auditing and model/prompt version integration.
