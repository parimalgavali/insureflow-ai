# Backend

Spring Boot backend foundation for InsureFlow AI.

## Current Status

Phase 0/1 creates the initial API application, Flyway schema, OpenAPI setup, and migration tests.

## Planned Responsibilities

- Core insurance domain schema.
- REST API foundation and OpenAPI documentation.
- Policy and claim workflows in later phases.
- Audit and governance persistence.
- Integration APIs after the core workflows exist.
